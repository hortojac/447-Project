iBuy is an e-commerce website where users can shop for various products.<br>
View website at: https://people.eecs.ku.edu/~j242h828/447-Project/

## Users
* View products by brand, category, search, or view all products.

## Admin Users
* View all products.
* Add a product.
* View all brands.
* Add a brand.
* View all categories.
* Add a category.
* View all users.
